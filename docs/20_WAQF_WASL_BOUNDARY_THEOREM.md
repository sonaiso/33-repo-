## §1. التأسيس: لماذا قبل الوزن؟

```
ليس كل كلمة = جذر + وزن + صيغة.

بل:
كل كلمة صرفية متمكنة أو مشتقة، بعد تحقق شروط الوقف والوصل والحدود،
يمكن أن تُولد من أصل + وزن + صيغة.

أما الحرف، والمبني، والإحالة، وبعض الأسماء الجامدة،
فلها مسارات حدودية قبل الوزن أو خارج الوزن.
```

**الاعتماد**: TH3 (الوزن حاجز لا مصدر)، P4 (لا معنى من الوزن وحده)

## §3. المبرهنة (TH6.5): الوقف والوصل وفق قانون الاقتصاد

### الصياغة

```
∀x ∈ LinguisticUnit:
    status(x) is determined by boundary_test at each level:

    CanStop(x, level) ⇔ BoundaryClosed(x, level)
    MustJoin(x, level) ⇔ RequiredComplement(x, level) ≠ ∅

    where level ∈ {PHONETIC, STRUCTURAL, FUNCTIONAL, SEMANTIC}
```

**الوقف اختبار إغلاق، لا اختبار معنى نهائي.**
**الوصل علامة حاجة بنيوية أو وظيفية أو دلالية.**

### البرهان (10 خطوات)

| الخطوة | البيان | المبرّر |
|--------|--------|---------|
| 1 | المقطع هو الحد الأدنى المكتمل للنطق | P31, §4 MCE-2 |
| 2 | اللفظ سلسلة مقاطع مرتبة | P31, BL-L0-04 |
| 3 | الدال لا يكون دالًا إلا بمراساة مع مدلول | P1 (Sound Primacy), def.R |
| 4 | الكلمة لا تكون كلمة إلا إذا أغلقت مسارًا لفظيًا/وظيفيًا | P32 |
| 5 | ليس كل إغلاق صوتي إغلاقًا بنائيًا | P31 ≠ P32 (مستويات منفصلة) |
| 6 | ليس كل إغلاق بنائي إغلاقًا إفاديًا | P32 ≠ P33 (مستويات منفصلة) |
| 7 | الوقف اختبار لمستوى الإغلاق، لا دليلًا على كمال المعنى | من 5 و6 |
| 8 | كل وحدة تحتاج غيرها لإتمام وظيفتها فهي موصولة في ذلك المستوى | P34 |
| 9 | قانون الاقتصاد يمنع صرف بنية أعلى قبل إغلاق الأدنى | P35, P36, P37 |
| 10 | **إذن: لا وزن قبل حد الكلمة، ولا كلمة قبل حد الدال، ولا معنى مكتمل قبل علاقة مغلقة** | P37, P38, TH3 |

## §4. أنواع الوقف

| النوع | السؤال | اختبار الإغلاق | مثال |
|-------|--------|---------------|------|
| الوقف الصوتي | هل يمكن النطق والتوقف؟ | `syllable_closed(x)` | زيدْ، مِنْ، هذا |
| الوقف البنائي | هل اكتملت بنية اللفظ/الكلمة؟ | `word_path_closed(x)` | رجل، كتب، هذا |
| الوقف الوظيفي | هل يعمل وحده أم يحتاج متعلقًا؟ | `function_self_sufficient(x)` | من ← تحتاج مجرورًا |
| الوقف الإفادي | هل تحقق معنى مكتمل؟ | `relation_closed(x)` | زيد قائم، قام زيد |

## §6. الحدود المُعرَّفة

### حد الوزن

```
WeightBoundary =
    licensed anchors
    + ordered slots
    + vowel distribution
    + added/omitted material
    + declared morphological function
    → يقف عند إنتاج صورة صرفية مرخصة
    → لا يتجاوز إلى معنى مستقل أو حكم نحوي أو إفادة
```

### حد الكلمة

```
WordBoundary =
    مقاطع مرخصة
    + ترتيب مغلق
    + دال مرسى
    + مسار وظيفي محدد (اسم/فعل/حرف/مبني/معرب)
    + قابلية وقف/وصل معلنة
```

### حد المبني

```
MabniBoundary =
    surface fixed (صورة ثابتة)
    + i'rab blocked (الإعراب ممنوع)
    + function/reference path declared
    + no derivational weight path unless rebuild
```

### حد المعرب

```
Mu'rabBoundary =
    word internally licensed
    + ending open to external operator
    + syntactic case/mood not fixed internally
    + final surface determined by relation
```

### حد الحرف

```
HarfBoundary =
    fixed surface
    + operator function
    + required operand (MustJoin obligatory)
    + no independent lexical closure
```

### حد المعنى المكتمل

```
CompleteMeaning =
    WordBoundary(x₁)
    ∧ WordBoundary(x₂)
    ∧ LicensedRelation(x₁, x₂)
    ∧ ReferenceClosed
    ∧ NoBlockingResidual
```

## §7. الصيغة الرمزية المغلقة

```
CanStop(x, level) ⇔ BoundaryClosed(x, level)

MustJoin(x, level) ⇔ RequiredComplement(x, level) ≠ ∅

CompleteMeaning(x) ⇔
    WordBoundary(x₁)
    ∧ WordBoundary(x₂)
    ∧ LicensedRelation(x₁, x₂)
    ∧ ReferenceClosed
    ∧ NoBlockingResidual
```

## §9. الخلاصة الحاكمة

```
الوقف والوصل هما برهان الحدود.

فما صح الوقف عليه صوتيًا صار لفظًا من جهة النطق،
وما صح الوقف عليه بنائيًا صار كلمة أو مبنيًا أو معربًا من جهة الصورة،
وما صح الوقف عليه وظيفيًا صار وحدة عاملة أو مستقلة،
وما صح الوقف عليه إفاديًا صار معنى مكتملًا.

أما ما احتاج غيره:
فهو موصول في المستوى الذي احتاجه.

لذلك:
لا وزن قبل حد الكلمة،
ولا كلمة قبل حد الدال،
ولا دال قبل مراساة مدلول،
ولا معنى مكتمل قبل علاقة مغلقة.
```
