## Authority

## Core Law

No Boolean as proof.
No ProofObject without trace.
No Trace without evidence.
No Evidence without scope, residual, and invalidator.

Required proof objects:
- MRKProof is required.
- IdentityProof is required.
- GateProof is required.
- BridgeProof is required.
- DomainProof is required.
- EvidenceProof is required.
- CoverageProof is required.

Forbidden substitutions:
- `identity_preserved: bool`
- `gate_passed: bool`
- `domain_proved: bool`
