## §2. المبرهنة السابعة (TH7): النظام التوليدي الصرفي

### الصياغة

```
∀R ∈ {R₃, R₄}, ∀W ∈ WeightPattern, ∀F ∈ Form:
    licensed(R) ∧ valid(W) ∧ defined(F) → G(R, W, F) = S
```

**حيث S صيغة سطحية وحيدة محددة.**

### البرهان (8 خطوات)

| الخطوة | البيان | المبرّر |
|--------|--------|---------|
| 1 | R مرخّص (3 أو 4 حروف صحيحة) | P20 |
| 2 | W يحدد القالب: (مواقع, بادئة, حشوة, لاحقة, تشديد) | P21 |
| 3 | F يحدد الهدف (ماضي, مضارع, اسم فاعل, ...) | P22 |
| 4 | G تأخذ (R, W, F) وتُنتج S | P22 (دالة كلية) |
| 5 | S وحيدة: نفس المدخلات تُنتج نفس المخرج | P30 (حتمية) |
| 6 | S لا تحمل معنى جديد — المعنى في R فقط | P29, TH1 |
| 7 | W وسيط يرخّص الظهور لا ينتج معنى | TH3, P9 |
| 8 | **إذن G(R,W,F) = S: نظام توليدي حتمي كامل** | P20–P30 |

### الصيغ المُولَّدة

| # | الهدف (F) | النمط | المثال (ك-ت-ب) |
|---|-----------|-------|---------------|
| 1 | الماضي | فَعَلَ | كَتَبَ |
| 2 | المضارع | يَفْعُلُ | يَكْتُبُ |
| 3 | اسم الفاعل | فَاعِل | كَاتِب |
| 4 | اسم المفعول | مَفْعُول | مَكْتُوب |
| 5 | المصدر | فِعَال | كِتَاب |
| 6 | جمع مذكر سالم | فَاعِلُونَ | كَاتِبُونَ |
| 7 | جمع مؤنث سالم | فَاعِلَات | كَاتِبَات |
| 8 | جمع تكسير | فُعَلَاء | كُتَبَاء |
| 9 | التصغير | فُعَيْل | كُتَيْب |
| 10 | التفضيل | أَفْعَلُ | أَكْتَبُ |

## §3. المبرهنة الثامنة (TH8): التعدي واللزوم

### الصياغة

```
∀W ∈ WeightPattern: T(W) = transitivity_class
    where T is determined by structural features of W
```

**التعدي ليس عشوائياً — هو دالة من بنية الوزن.**

### البرهان (6 خطوات)

| الخطوة | البيان | المبرّر |
|--------|--------|---------|
| 1 | W يحدد البنية: (بادئة, حشوة, تشديد) | P21 |
| 2 | التشديد (Form II) = تكثيف = تأثير خارجي = تعدي | P23 |
| 3 | الألف الوسطية (Form III) = تبادل = مفعول مشارك = تعدي | P23 |
| 4 | الهمزة البادئة (Form IV) = تسبيب = مسبَّب = تعدي | P23 |
| 5 | النون البادئة (Form VII) = انفعال = لا مفعول = لزوم | P23 |
| 6 | **إذن T(W) محدد بالبنية لا بالسماع** | P23, P21 |

### جدول التعدي

| الوزن | الآلية | الحكم | عدد المفاعيل |
|-------|--------|-------|-------------|
| فَعَلَ (I) | لا زيادة | متغير | 0–1 |
| فَعَّلَ (II) | تشديد | متعدٍ | 1 |
| فَاعَلَ (III) | ألف وسطية | متعدٍ | 1 |
| أَفْعَلَ (IV) | همزة بادئة | متعدٍ | 1 |
| تَفَعَّلَ (V) | تاء بادئة | متغير | 0–1 |
| تَفَاعَلَ (VI) | تاء + ألف | متغير | 0 |
| اِنْفَعَلَ (VII) | نون بادئة | لازم | 0 |
| اِفْتَعَلَ (VIII) | تاء داخلية | متغير | 0–1 |
| اِسْتَفْعَلَ (X) | سين + تاء | متعدٍ | 1 |

## §4. المبرهنة التاسعة (TH9): المعتل

### الصياغة

```
∀R containing weak_consonant at position p:
    M(R, p, F) = deterministic_transformation
```

**الإعلال ليس استثناءً — هو قاعدة حتمية من الموقع.**

### البرهان (5 خطوات)

| الخطوة | البيان | المبرّر |
|--------|--------|---------|
| 1 | WEAK = {و, ي, ء} مجموعة مغلقة | P24 |
| 2 | الموقع يحدد الصنف: (0→مثال, 1→أجوف, 2→ناقص) | P25, P26, P27 |
| 3 | لكل (صنف, هدف) قاعدة تحويل واحدة | P28 |
| 4 | التحويل حتمي: نفس المدخل → نفس المخرج | P28, P30 |
| 5 | **إذن الإعلال نظام قواعدي لا استثنائي** | P24–P28 |

### جدول التحويلات

| الصنف | الموقع | الماضي | المضارع |
|-------|--------|--------|---------|
| مثال | الأول (F) | همزة بادئة (Form IV) | حذف الواو |
| أجوف | الثاني (A) | قلب إلى ألف | حفظ كحركة طويلة |
| ناقص | الثالث (L) | ألف مقصورة | قلب إلى ياء |
| لفيف مقرون | A+L | ألف مقصورة | قلب إلى ياء |

## §5. الخلاصة الرياضية

```
المُدخل:
    R = (c₁, c₂, c₃) ∈ CONSONANTS³     [الجذر]
    W = (pos, pre, inf, suf, gem) ∈ WEIGHT  [الوزن]
    F ∈ {PAST, PRESENT, ACT_PART, ...}    [الهدف]

الدالة التوليدية:
    G: R × W × F → S                     [TH7]
    T: W → {LAZIM, MUTAADDI, VARIABLE}    [TH8]
    M: R × pos × F → S_transformed       [TH9]

الخصائص:
    1. G كلية (total): كل مدخل صالح يُنتج مخرجاً
    2. G حتمية (deterministic): نفس المدخل → نفس المخرج
    3. G لا تُنتج معنى: meaning(G(R,W,F)) = meaning(R)
    4. T دالة: التعدي محدد بالبنية لا بالسماع
    5. M حتمية: الإعلال قاعدة لا استثناء
```
