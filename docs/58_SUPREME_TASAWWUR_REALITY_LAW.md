## The Supreme Law

```text
الحكم على الشيء فرع عن تصوره.
والتصور فرع عن إحالة.
والإحالة فرع عن افتقار.
والافتقار فرع عن أثر.
والأثر فرع عن واقع.

فلا حكم بلا تصور.
ولا تصور معتبر بلا معقولية.
ولا معقولية بلا إحالة ومجال وحامل أو تحول أو ربط.
ولا إحالة بلا أثر أو معرفة سابقة أو قرينة.
ولا أثر معتبر بلا انتقال محفوظ.
ولا انتقال محفوظ بلا واقع سابق أو واقع مفترض بقدر الدليل.
```

## Declaration

> The Supreme Tasawwur-Reality Law establishes that judgment (حكم) does not begin
> from language, nor from the carrier, nor even from cognition — it begins from:
>
> **Reality (واقع) → Effect (أثر) → Attention (تنبه) → Need (افتقار) → Reference (إحالة) → Conception (تصور)**
>
> This is the chain of possibility (سلسلة الإمكان) that precedes reason.

## The 21 Layers

### Layer 0 — Reality Layer (طبقة الواقع)

```text
واقع كيان، واقع تحول، واقع ربط، واقع أثر.
```

Reality is not entities alone. It contains:

| Type | Meaning |
|------|---------|
| واقع كيان | Something with existence or determination |
| واقع تحول | Change, motion, action, transition, state |
| واقع ربط | Relation between entity and transformation, or entity and entity |
| واقع أثر | What reality leaves in sense, memory, or evidence |

**Formula**: Reality = Entity + Transformation + Linking + Effect.

---

### Layer 1 — Effect Transfer Layer (طبقة انتقال الأثر)

```text
انتقال أثر من الواقع إلى جهاز صالح.
```

An effect does not become knowledge merely by existing. It must transfer to a capable receiver.

Examples:
- Physical sound → wave → ear/receptor → neural/perceptual effect
- Light → reflection → eye → visual effect

The effect is a primary reference from reality to a receptive capacity.

---

### Layer 2 — Receptive Brain Layer (طبقة الاستقبال)

```text
تلقي، تمييز، حفظ، مقارنة.
```

| Operation | Function |
|-----------|----------|
| تلقي (Reception) | Entry of the effect |
| تمييز (Discrimination) | Isolating it from others |
| حفظ (Retention) | Preserving its trace |
| مقارنة (Comparison) | Linking it to prior traces |

This is the pre-judgment, pre-proof layer — a layer of capacity.

---

### Layer 3 — Iftiqar-Attention Layer (طبقة الافتقار والتنبه)

```text
افتقار يفتح التنبه وطلب الإحالة.
```

Need (افتقار) means: the appearance of a gap in explanation that drives the apparatus
to seek a reference.

The intellect moves only when there appears:
- An effect needing a source
- A difference needing explanation
- An unknown needing a carrier
- A transformation needing a cause
- A linking needing parties

**Chain**: Need → Attention → Request for Reference.

---

### Layer 4 — Pre-Conceptual Reference Layer (طبقة الإحالة القبلية)

```text
رد الأثر إلى مصدر محتمل.
```

This is not yet full conception. It is a pre-conceptual reference — probabilistic candidacy:

- This effect is perhaps a sound.
- This sound is perhaps human.
- This human sound is perhaps Arabic.
- This drawing is perhaps a letter.
- This utterance is perhaps a name.
- This name is perhaps a carrier.
- This transformation is perhaps a verb.

This is a layer of probabilistic candidates.

---

### Layer 5 — Tasawwur Layer (طبقة التصور)

```text
تصور حامل أو تحول أو ربط.
```

Conception is the first considered intellectual closure. It is not always conception of an entity alone:

| Type | Example |
|------|---------|
| تصور حامل (Carrier conception) | Stone, human, moon |
| تصور تحول (Transformation conception) | Writing, falling, movement, combustion |
| تصور ربط (Linking conception) | In, from, if, because, with, not |

**Separating Rule**: No relation without conceived parties. No judgment without an understood relation.

---

### Layer 6 — Prior Classified Knowledge Layer (طبقة المعرفة السابقة المصنفة)

```text
معلومات سابقة عن الكيان والحقيقة والخواص والتحولات والربط والأثر والمجال.
```

This layer does not judge alone — it classifies.

Example: A circular luminous visual effect in the sky → summons prior knowledge:
moon / sun / lamp / reflection / celestial body.

**Law**: `PriorKnowledge → CandidateSet` (never `PriorKnowledge → Truth`).

---

### Layer 7 — Carrier-Identity Layer (طبقة الحامل والهوية)

```text
حامل يحفظ الهوية داخل مجال إحالة.
```

After reference, conception, and prior knowledge, a carrier forms:

```text
Carrier = identity-preserved carrier within a domain, with rank, and with residuals.
```

**Laws**:
- No carrier without pre-conceptual reference.
- No carrier without a domain.
- No carrier without identity preservation.

Example: "عين" (eye/spring/spy/essence) does not become a determined carrier
until a domain is specified:
- عين الإنسان → organ
- عين الماء → spring
- عين العدو → spy
- عين الشيء → its very self

---

### Layer 8 — Genus-Essence-Property Layer (طبقة الجنس والحقيقة والخواص)

```text
جنس، نوع، صنف، حقيقة، خواص، أصل خواص، محل خواص.
```

The ordering:

```text
Carrier → Genus → Species → Class → Essence → Properties → Source of Properties → Locus of Properties
```

| Layer | Function |
|-------|----------|
| الجنس (Genus) | Classifies identity |
| النوع (Species) | Specifies within genus |
| الصنف (Class) | Fixes a sub-branch within species |
| الحقيقة (Essence) | Establishes "what makes it what it is" |
| الخواص (Properties) | Reveals the carrier's capacities |
| أصل الخواص (Source of properties) | Intrinsic, accidental, manufactured, figurative |
| محل الخواص (Locus of properties) | Where does the property inhere? In the essence? In a part? In a relation? |

---

### Layer 9 — Attribute-Transformation Layer (طبقة الصفة والتحول)

```text
صفة، مصدر تحول، فاعلية، مفعولية، سببية، مسببية.
```

Attributes are not attributed to the carrier directly — they are traced to a transformation source:
- كاتب ← كتابة (writer ← writing)
- مكسور ← كسر (broken ← breaking)
- حارّ ← حرارة (hot ← heat)

Tests applied:

| Test | Question |
|------|----------|
| الفاعلية (Agency) | Does the attribute have an agent? |
| المفعولية (Patiency) | Does it fall on a locus? |
| السببية (Causation) | Is the carrier the cause of transformation? |
| المسببية (Being caused) | Is the carrier the effect of a prior transformation? |
| المطاوعة (Receptivity) | Did it accept an effect? |
| المشاركة (Participation) | Does it require two parties? |
| المكان (Place) | Does it require a location? |
| الزمان (Time) | Does it require a time? |

---

### Layer 10 — Weight Witness Layer (طبقة الوزن الشاهد)

```text
الوزن يشهد على فرع مرخص، لا يخلق الأصل.
```

The correct order:

```text
Transformation source → Branch → Weight witness
```

Never: `Weight → Meaning`.

Example:
- كتابة → فرع فاعلي → كاتب (on pattern فاعل)
- كتابة → فرع مفعولي → مكتوب (on pattern مفعول)
- فتح → فرع آلي → مفتاح (instrument)

Weight establishes morphological capacity — not judgment, not reality.

---

### Layer 11 — Operator-Linking Layer (طبقة الأداة والربط)

```text
أداة تضبط الربط والمنع والشرط والسبب والإحالة.
```

Operators are not ordinary entities or attributes. They are linking gates:

| Operator | Gate |
|----------|------|
| و | AND |
| أو | OR |
| لا / لم / لن | NOT |
| لا...ولا | NOR |
| إن / إذا | IF-THEN |
| إلا | AND-NOT |
| إنما / لا...إلا | ONLY-IF |
| لأن / كي / لام التعليل | CAUSE |
| ثم / فـ | SEQUENCE |
| من / إلى | START/END |
| في / على | CONTAINMENT/SUPERIORITY |
| هذا / هو / الذي | DISCOURSE REFERENCE |

The operator determines: Is the linking conjunction? Blocking? Conditional? Causal? Exceptional? Referential?

### Layer 12 — Phono-Lafz Layer (طبقة الصوت واللفظ)

```text
حالات صوتية/كتابية مرتبة ومركبة للإحالة.
```

Two paths must be distinguished:

**a. Incoming path (من الواقع إلى اللفظ)**:
```text
Physical event → Sound → Human sound → Arabic sound → Articulation point and feature
→ Letter/Vowel mark → Script/Unicode → Utterance → Signifier
```

**b. Outgoing path (من التصور إلى اللفظ)**:
```text
Conception → Choice of signifier → Single utterance → Word → Composition
```

| Aspect | Function |
|--------|----------|
| Incoming | Analyzing a phonetic/written trace into an utterance |
| Outgoing | Converting a conception into a signifying utterance |

---

### Layer 13 — Lexical Closure Layer (طبقة إقفال المفرد)

```text
إقفال اللفظ المفرد: كيان، صفة، مصدر، تحول، أداة، علم، مبني، دخيل، بقايا.
```

Closure of the singular utterance is a condition of possibility for the word.

Closing a singular means determining whether it is:
- Entity (كيان)
- Attribute (صفة)
- Source/Masdar (مصدر)
- Transformation (تحول)
- Operator (أداة)
- Proper name (علم)
- Indeclinable (مبني)
- Foreign (دخيل)
- Residual (بقايا)

Only then is it qualified to become a word.

---

### Layer 14 — Word Contract Layer (طبقة الكلمة)

```text
المفرد المغلق يصير كلمة: اسم، فعل، حرف.
```

| Word type | Condition of possibility |
|-----------|--------------------------|
| الاسم (Noun) | Carrier, attribute, masdar, proper name, or a meaning capable of predication/annexation |
| الفعل (Verb) | Transformation bound to time, aspect, or predication |
| الحرف (Particle) | Linking operator that does not have independent carrier-capacity |

---

### Layer 15 — Syntax-Composition Layer (طبقة التركيب)

```text
الكلمة تدخل التركيب عبر العامل والمعمول.
```

After word closure, words enter the contract:
- عامل (Governor): refers to a demand
- معمول (Governed): fills that demand
- رابط (Linker): operator connecting parties
- رتبة (Position): syntactic rank
- علامة (Case marking): trace of the relation
- أثر إعرابي (Syntactic effect): manifestation of the relation

Governor and governed are not merely syntactic positions — they are parties of compositional reference.

---

### Layer 16 — Relation Layer (طبقة النسب)

```text
إسناد، تضمين، تقييد، فاعلية، مفعولية، سببية، مسببية.
```

Major relations:

| Relation | Meaning |
|----------|---------|
| إسنادية (Predicative) | Bearing a predicate or verb upon a carrier |
| تضمينية (Inclusive) | Embedding a meaning within a meaning, or part within whole |
| تقييدية (Restrictive) | Narrowing the domain of a carrier or attribute |
| فاعلية تركيبية (Compositional agency) | Attributing a verb to an agent |
| مفعولية تركيبية (Compositional patiency) | A verb falling upon a patient |
| سببية تركيبية (Compositional causation) | Linking a verb to a cause |
| مسببية تركيبية (Compositional being-caused) | Linking an effect to a prior cause |

These relations are the condition of possibility for case markings.

---

### Layer 17 — Discourse Reference Layer (طبقة الإحالة الخطابية)

```text
ضمير، إشارة، موصول، سياق، مقام، منطوق، مفهوم.
```

After composition, another type of reference appears — discourse reference:

| Element | Type of reference |
|---------|-------------------|
| الخبر (Declarative) | Reference to truth/falsity capability |
| الإنشاء (Performative) | Reference to request, command, prohibition, interrogation, wish |
| المقام (Context) | Reference to circumstance of use |
| السياق (Cotext) | Reference to preceding and following |
| المتكلم (Speaker) | Reference to intention or address |
| المخاطب (Addressee) | Reference to capacity for understanding and reception |
| المنطوق (Explicit) | What the utterance closed within the boundary of saying |
| المفهوم (Implied) | What is understood outside the boundary of the explicit, by license |
| المعقول (Rational) | What conception, relation, and evidence accept |

**Law**:
- No mafhum without a preserved mantuq.
- No mafhum without an origin.
- No origin without branch-capacity.
- No branch-capacity without an inspected preventer.
- No rational meaning without reference and domain.

---

### Layer 18 — Ifadah Layer (طبقة الإفادة)

```text
إغلاق النسبة بين تصورات معقولة.
```

Not every sentence communicates by merely containing words. Communication requires:
- Conceived parties
- Closed relation
- Valid linking
- Absence of preventer
- Sufficient context

Example: "الرجل كاتب" (The man is a writer) is communicative because:
- الرجل = conceived carrier
- كاتب = conceived attribute with transformation source
- The relation = bearing the attribute upon the carrier

---

### Layer 19 — Hukm Layer (طبقة الحكم)

```text
حكم بدليل.
```

Communication does not suffice. "الرجل كاتب" is communication.
But the judgment "هذا الرجل كاتب فعلًا" (This man is actually a writer) requires evidence:
- Witnessed writing
- Has books
- Works in writing
- A credible witness testified

**Law**: No judgment without evidence.

---

### Layer 20 — Tanzil Layer (طبقة التنزيل)

```text
تنزيل بتحقيق مناط.
```

Application requires verification of the applicable basis (تحقيق المناط).

Example:
- "Every writer excels at expression" (general rule).
- "Zayd is a writer" (particular claim).

Before applying the rule to Zayd:
- Is Zayd actually a writer?
- What does "writer" mean here? Professional? Occasional? Instrumental? Textual?
- Does the rule encompass this type?

This is تحقيق المناط (verification of the basis of application).

## General Transition Law (قانون الانتقال العام)

Every transition in this chain requires a licensed reference:

```text
Transition(A → B) is licensed iff:
    Reference(A, B)
    AND Domain(A, B)
    AND CarrierOrOperationPossible(B)
    AND IdentityPreserved(B)
    AND Condition(A, B)
    AND Cause(A, B)
    AND NOT Preventer(A, B)
    AND NOT InvalidatingDifference(A, B)
    AND Rank(B) <= EvidenceRank
    AND ResidualsDeclared
```

In Arabic:

```text
لا انتقال بلا إحالة.
ولا إحالة بلا مجال.
ولا مجال بلا حامل أو تحول أو ربط.
ولا ترخيص بلا شرط وسبب.
ولا عبور مع مانع أو فرق قادح.
ولا رتبة فوق الدليل.
ولا بقايا مخفية.
```
