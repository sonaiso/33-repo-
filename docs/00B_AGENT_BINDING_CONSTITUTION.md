# Agent Binding Constitution — `docs/00B_AGENT_BINDING_CONSTITUTION.md`

## Authority
Derived from `docs/00_MAQOOL_CONSTITUTION.md §5` and `docs/00A_CONSTITUTIONAL_PROGRAMMING_AMENDMENT.md`.

## Principle

Agent proposes.
Kernel verifies.
Coverage audits.
Constitution governs.

## Role Boundaries

The agent is a proposer, not a judge.
The agent may submit intent, candidate transitions, and evidence objects.
The agent may not convert declaration into proof.

## Non-Delegable Kernel Powers

The agent may not grant MRK.
The agent may not grant Bridge.
The agent may not raise rank.
The agent may not close Coverage.
The agent may not issue final meaning.
The agent may not open B6, L2, or L3.

If the agent cannot provide required ProofObject contracts, output must be:
`BLOCKED_MISSING_PROOF`.
