## Boundary Separation Law

```text
النسبة ليست العامل،
والعامل ليس الطرف،
والعلامة الإعرابية ليست المقام.
```

- Nisba is an independent typed binding structure.
- Operator is a possible activation mechanism in some relation types.
- Case marker is a formal realization effect within syntax.
- Maqam is a discourse frame that constrains interpretation and speech force.

None of these layers substitutes for another layer.

- arity satisfied;

- no defeating difference;
- no blocking preventer.


operator.
